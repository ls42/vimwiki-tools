# vimwiki-tools

install via pip
```sh
pip install vimwiki-tools
```

generate index
```sh
vimwiki gen-index ~/wiki/linux --write

vimwiki gen-index ~/wiki/linux --output-type html
```

convert to markdown
```sh
vimwiki 2markdown ~/wiki/programming --output-extension wiki
```

view stats
```sh
vimwiki stats ~/wiki
```

for more options
```sh
vimwiki -h
```
